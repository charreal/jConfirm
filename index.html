jConfirm - jQuery Plugin

jQuery Confirmation Tooltips

jConfirm v.2

Want to confirm a delete but hate the default javascript confirm()? Say hello to jConfirm, the solution to all your woes.

Test It GitHub

Beautiful

You can customize:

  • The Design
  • Message Text
  • Button Labels
  • Callback
  • Attributes Saved for Callback

Example Code (actual code from demo above):

<!-- Includes the CSS -->
<link rel="stylesheet" type="text/css" href="jConfirm-v2.css">

<!-- Create buttons -->
<a href='#' class="btn btn-lg btn-success jConfirm" role="button" itemType='product' itemId='1'>Test It</a>
<a href='http://github.com' class="btn btn-lg btn-primary jConfirmTwo" role="button">GitHub</a>

<!-- Includes the jQuery and the JS -->
<script src='https://code.jquery.com/jquery-2.0.3.min.js'></script>
<script src='jConfirm-v2.js'></script>

<!-- Attach jConfirm to your buttons -->
<script>
$('.jConfirm').jConfirm({ 
message: 'You sure you to delete this?', 
confirm: 'YUPPERS', 
cancel: 'NO WAY!', 
attributes: ['itemType', 'itemId'], 
callback: function(attrs){ 
alert('According to the link you clicked...\r\nType = '+attrs.itemType+' & ID = '+attrs.itemId);
} 
}); 
$('.jConfirmTwo').jConfirm({ 
message: "You sure you're ready to use our badass plugin?", 
confirm: "Let's GO!", 
cancel: "Sorry, I'm a Loser...", 
attributes: ['href'], 
callback: function(attrs){
window.location.href = attrs.href; 
} 
}); 
</script>
